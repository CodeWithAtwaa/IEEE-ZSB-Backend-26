# Research Questions

## UNION vs UNION ALL
* Both used to combine the result-set of two or more  `SELECT` statements.

### UNION 
* Remove Duplicates
* return unique record
* like as `DISTINCT`
* slower

### UNION ALL
* contain Duplicates
* Simply appends the second result set to the first
* Faster
----------


## Subquery vs JOIN

### JOIN
* is used to combine Rows from two or more tables, based on a related column between them.
* can use + DML
* combine columns ( دا اللي بيزيد )

#### Has many Type of join 
- left join
- right join
- full join
- inner join

### Subquery
-  query nested query.
- Take output from query as input for another query. 
-  can use  +  DML.
- can write any where.

#### Sub-Query Categories
* Dependancy
* Result Type
* Location
